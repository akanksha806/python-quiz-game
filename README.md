# Python Quiz Game

This is a simple Python quiz game created for practice.

## Features
- Multiple choice questions
- Score calculation
- User input validation

## Concepts Used
- while loop
- if-else conditions
- lists
- string handling

## How to Run
1. Download the project
2. Run the Python file using Python 3
3. Answer the questions in the terminal

## Purpose
This project was made to practice Python basics and improve logic building.
